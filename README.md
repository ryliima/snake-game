# 🐍 Snake Game

## 📖 Sobre o Projeto

O **Snake Game** é uma versão moderna do clássico jogo da cobrinha desenvolvida utilizando tecnologias web puras.

O objetivo é simples:

- 🍎 Comer a comida espalhada pelo mapa.
- 📈 Aumentar a pontuação.
- 🐍 Fazer a cobrinha crescer.
- 🚫 Evitar colisões que encerram a partida.

O projeto foi criado com foco em aprendizado de lógica de programação, manipulação do DOM e desenvolvimento de jogos utilizando HTML, CSS e JavaScript.

---

## 🎮 Demonstração

### Tela Inicial
- Botão para iniciar a partida.
- Interface simples e intuitiva.

### Durante o Jogo
- Controle da cobrinha através das setas do teclado.
- Sistema de pontuação em tempo real.
- Registro de recorde (Best Score).
- Geração aleatória de alimentos.

### Game Over
- Exibição da pontuação final.
- Opção para reiniciar rapidamente.
